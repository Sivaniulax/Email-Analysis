Email-Analysis

Data visualization project performing EDA, visualization using past and live data on  the gmail server.

This is an data analysis and visualization project which perform data cleaning ,visualization and analysis on the google server (gmail).

The main objective of the project is to make the data cleaning easier in the server rather than performing manual deletion, I implemented code to clean data by grouping them into categories and to make the cleaning process easier.
